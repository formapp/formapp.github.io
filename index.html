Flows - The core of FlowApp

flows

Using cutting edge technology developed by Facebook, FlowApp gives you the power to create micro-apps, called Flows.

Create flows to manage all operations related to your business.

Define the software that you want to use.

Each micro-app, or a Flow is made up of three things:

1. Fields - to define the data structure of a process

2. Statuses - to represent the state of each item in a process

3. Steps - to represent how an item moves through a process

These three are combined to create a Flow, which can be used to manage a business process.
